Question-1: What are the different ways to select an element in the DOM?
There are many ways to select any elements in DOM, such as
getElementsByClassName: It returns multiple elements that have the same class.
getElementById: It returns a single element that has a specific ID.
getElementsByTagName: It returns multiple elements that have the same tag name.
querySelector: It returns the first element that matches a specific CSS selector.
querySelectorAll: It returns all elements that match a specific CSS selector as a static NodeList.
Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML contains the HTML content inside an element, including tags.
innerText contains the visible text, considering styles like visibility.
textContent contains all the text, including hidden text, without considering styling or formatting.
Question-3: What is event delegation in the DOM?
Event delegation is when attaching a single event listener to a parent element instead of individual child elements and still it detects which child triggered the event through event bubbling.
Question-4: What is event bubbling in the DOM?
Event bubbling is a technique in the DOM where an event starts from the target element and then goes through to its parent elements. This allows multiple parent elements to listen for the same event that happened in their children. This helps in enabling event delegation and making the handling of events more efficient.
Question-5: How do you create, add, and remove elements using JavaScript?
To create, add, and remove elements using JavaScript, at first an element has to be created by "document.createElement('tagName')", where 'tagName' is the type of element, such as 'div', 'p', etc.
After that, to add the element to the document, "appendChild()" is used to append it to an existing parent element in the DOM. For example, "parentElement.appendChild(newElement)" will add the newly created element as the last child of the parent element.
When it is time to remove an element, "removeChild()" is used on the parent element to remove the desired child. For instance, "parentElement.removeChild(newElement)" will remove the created element from its parent. These methods allow to dynamically manipulate the DOM, adding or removing elements as needed.
