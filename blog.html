Question-1: What are the different ways to select an element in the DOM?

There are many ways to select any elements in DOM, such as
getElementsByClassName: It helps to find multiple elements that have the same class.
getElementById: It helps to find a single element that has a specific ID.
getElementsByTagName: It helps to find multiple elements that have the same tag name (e.g., div, p).
querySelector: It helps to find the first element that matches a specific CSS selector (ID, class, tag, etc.).
querySelectorAll: It helps to find all elements that match a specific CSS selector, returning a static NodeList.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gives you or sets the HTML content inside an element, including tags. innerText gets or sets the visible text, considering styles like visibility. textContent gives or sets all the text, including hidden text, without considering styling or formatting. textContent is faster than innerText because it doesn't account for the layout.

Question-3: What is event delegation in the DOM?

Event delegation is when you attach a single event listener to a parent element instead of individual child elements. The event listener detects which child triggered the event through event bubbling, improving performance, especially with many dynamic elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event starts from the target element and then propagates upwards to its parent elements. This allows multiple parent elements to listen for the same event, enabling event delegation and making the handling of events more efficient.

Question-5: How do you create, add, and remove elements using JavaScript?

To create, add, and remove elements using JavaScript, you can start by creating an element with `document.createElement('tagName')`, where 'tagName' is the type of element you want to create, such as 'div', 'p', etc. After that, to add the element to the document, you can use `appendChild()` to append it to an existing parent element in the DOM. For example, `parentElement.appendChild(newElement)` will add the newly created element as the last child of the parent element. If you want to remove an element, you can use `removeChild()` on the parent element to remove the desired child. For instance, `parentElement.removeChild(newElement)` will remove the created element from its parent. These methods allow you to dynamically manipulate the DOM, adding or removing elements as needed.
